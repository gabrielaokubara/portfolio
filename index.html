Sistema Salus - Página do Visitante

A pedido da Divisão de Vigilância Epidemiológica da Secretaria Municipal de Saúde de Bauru, o Sistema Salus foi desenvolvido com o intuito de informatizar o dados de doenças, que eram armazenados anteriormente em tabelas .DBF, e com isso possibilitar o acesso remoto, e web que dispensa a instalação.

Mais funcionalidades, porém, foram implementadas ao decorrer do projeto, tais como, site de divulgação, contato com a população, entre outras.

Sistema Salus - Interface Administrativa

Na Interface Administrativa, o usuário, funcionário da Secretaria de Saúde de Bauru, tem o acesso facilitado dos dados por meio de tabelas, mapas e gráficos, podendo selecionar filtros de doenças específicas, de perfis de pacientes como sexo e idade, e de períodos do ano. Além, de gerar relatórios automáticos, e também publicar no site para a população, alertas de epidemia, como prevenir, entre outras informações.


Projeto Marvel - Lista de Heróis (carregamento por API)

Desafio criado pela ez.Devs, para o recrutamento front-end.

Nele, foi criada uma key no Marvel Developer Portal que permite o acesso aos arquivos jsons com as informações dos heróis atualizadas. Com o Ajax, é lido o .json e extraídos os dados que são impressos no HTML.
